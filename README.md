# About

This project is a clone of the famous [Pong](https://en.wikipedia.org/wiki/Pong) published by Atari.

It is intended for testing purposes of the [olcPixelGameEngine](https://github.com/OneLoneCoder/olcPixelGameEngine) only.

# Requirements

No external requirements needed. The only requirement is the olcPixelGameEngine, which is already packed with the repository.

# Build

```
git clone https://github.com/ltoenning/PingBallX.git
cd PingBallX
mkdir build && cd build
cmake ..
make -j 
```
